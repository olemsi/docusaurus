# sdk_go

sdk_go — это набор средств разработки для взаимодействия веб-сервисов, разработанных на Go, с платежной платформой Some Platform при проведении оплаты через Payment Widget. В этом разделе представлена информация о работе с sdk_go с примерами кода на языке программирования Go.

sdk_go совместим с Go версии 1.8 или выше и доступен для загрузки на GitHub по следующей ссылке: sdk_go_github.
