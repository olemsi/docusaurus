## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей с использованием платежного метода Mobile Money:

- Для проведения платежей доступен сервис оператора мобильной связи HelloPaisa.
- Валютой платежа может быть только.references/ru/currencies/TZS.dita

## Оплата через сервисы мобильных операторов

Для отображения страницы оплаты с предварительно выбранным оператором мобильной связи или страны пользователя помимо параметратакже передайте в запросе объектс параметрами:

- — указатель предварительного выбора оператора мобильной связи, значение —;`1`
- — код страны пользователя в формате ISO 3166-1 alpha-2, значение для Танзании —;`TZ`
- — идентификатор оператора мобильной связи, значение для оператора HelloPaisa —.`HELLOPAISA`

## Выплата через сервисы мобильных операторов

Выплата с использованием метода Mobile Money проводится через сервис оператора мобильной связи HelloPaisa. Чтобы провести выплату через сервис HelloPaisa, передайте в запросе идентификаторв параметре.`HELLOPAISA`
