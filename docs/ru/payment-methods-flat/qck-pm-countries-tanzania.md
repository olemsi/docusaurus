## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей с использованием платежного метода Mobile Money:

- Для проведения платежей доступен сервис оператора мобильной связи HelloPaisa.
- references/ru/currencies/TZS.ditaВалютой платежа может быть только.

## Оплата через сервисы мобильных операторов

force_payment_methodpayment_methods_optionsДля отображения страницы оплаты с предварительно выбранным оператором мобильной связи или страны пользователя помимо параметратакже передайте в запросе объектс параметрами:

- enable_mobile_service`1`— указатель предварительного выбора оператора мобильной связи, значение —;
- selected_country`TZ`— код страны пользователя в формате ISO 3166-1 alpha-2, значение для Танзании —;
- selected_operator`HELLOPAISA`— идентификатор оператора мобильной связи, значение для оператора HelloPaisa —.

## Выплата через сервисы мобильных операторов

`HELLOPAISA`account.service_providerВыплата с использованием метода Mobile Money проводится через сервис оператора мобильной связи HelloPaisa. Чтобы провести выплату через сервис HelloPaisa, передайте в запросе идентификаторв параметре.
