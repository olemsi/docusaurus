# Интерфейс

Интерфейс Dashboard разбит на тематические разделы, в каждом из которых представлены свои представления данных и инструменты для работы. Многие элементы в разделах являются типовыми, а большинство задач решается с использованием реестров. Для поиска данных и ряда общих операций используется основное меню в верхней части интерфейса, а для перехода между разделами — панель навигации в левой части интерфейса.

В этом разделе представлены краткие сведения о работе с основным меню, разделами и реестрами.
