# Как узнать обменный курс валюты

## Обзор

При работе с платежной платформой Some Platform вы можете в любой момент уточнить обменный курс для интересующей вас валютной пары. Данный раздел описывает, как составить и отправить запрос на получение информации об обменном курсе и как обработать полученный ответ.
