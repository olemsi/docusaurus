# Интеграция с использованием CMS-систем

Для сайтов, созданных на базе ряда CMS-систем, возможна интеграция платежной страницы Some Platform с использованием плагинов. Some Platform поддерживает плагины к следующим CMS:

Если в списке нет требуемой CMS, обратитесь в службу технической поддержки Some Platform
            support@example.com

- InSales
- Tilda
- Битрикс 24
- Magento
- OpenCart 2.3
- OpenCart 3.0
- osCommerce
- PrestaShop
- WordPress
- 1-С Битрикс
