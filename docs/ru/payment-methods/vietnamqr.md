## Сценарии использования

Проведение оплат с использованием метода title может выполняться как с перенаправлением пользователей к сервису провайдера, так и без перенаправления. В любом из вариантов пользователь должен отсканировать QR-код мобильным приложением банка. Обратитесь к курирующему менеджеру Some Platform для уточнения схемы работы.

Детальные сведения о том, что необходимо делать со стороны мерчанта для проведения платежей, а также о том, что можно использовать для анализа информации о проведенных платежах и операциях, представлены в следующих разделах этой статьи.
