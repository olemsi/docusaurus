# Использование плагина cms_opencart23_plugin для CMS cms_opencart23
        cms_opencart23_version

В данной инструкции описана работа с плагином cms_opencart23_plugin версии cms_opencart23_plugin_version. Этот плагин позволяет использовать платежные решения Some Platform на сайтах, созданных на базе CMS cms_opencart23 версии cms_opencart23_version.

Скачать плагин cms_opencart23_plugin можно.

## Схема работы

Взаимодействие с платежными решениями Some Platform осуществляется через платежную страницу.

Запросы на открытие платежной страницы формируются автоматически (с помощью плагина) при выборе cms_paymentmethod в методах оплаты.

Если по какой-либо причине платеж не прошел, у пользователя есть возможность повторно ввести данные на платежной странице.

По результатам проведения платежей на сайт отправляются оповещения, которые также автоматически обрабатываются с помощью плагина. В соответствии с результатом платежа изменяется:статус заказа

- — платеж не проведен.**Failed status**
- — платеж проведен.**Success status**
- — ожидание проведения платежа.**Pending status**
- — выполнен возврат средств до закрытия, в котором был проведен платеж.**Reversed status**операционного дня
- — выполнен возврат средств после закрытия операционного дня, в котором был проведен платеж.**Refunded status**
