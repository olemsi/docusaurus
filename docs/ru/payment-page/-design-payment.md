# Страница выбора платежного метода без сохранения данных

## Общая информация

Страница содержит перечень доступных платежных методов.

На странице можно менять оформление элементов интерфейса. Часть из них обязательно должна присутствовать на форме, часть может отсутствовать.

## Обязательные элементы

- Логотипы платежных методов. Для замены логотипов по умолчанию необходимо предоставить логотипы всех требуемых платежных методов в стандартном виде и при наведении указателя мыши на них. Порядок отображения платежных методов и групп платежных методов можно менять, но нет возможности менять состав групп платежных методов.
- Названия платежных методов.

## Дополнительные элементы

Сообщение «Все платежные методы».
