# Сохранение платежных данных пользователя

Для улучшения пользовательских сценариев при работе с Payment Widget предусмотрена возможность сохранять платежные данные и в дальнейшем использовать их без необходимости повторного ввода. В рамках этой возможности платежные данные могут быть сохранены при проведении оплат и проверке действительности платежных инструментов. Сохраненные данные отображаются в платежной форме на странице выбора способа оплаты вместе с данными платежных карт, на основании которых выполнялось формирование токенов.
