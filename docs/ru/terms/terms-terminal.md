# Терминал

Терминалы могут формироваться при встраивании индивидуальных текстовых формулировок в платежную форму, изменении ее дизайнаи других изменениях типового оформления Payment Widget. Формированием терминалов занимаются специалисты технической поддержки и предоставляют мерчанту идентификатор для каждого терминала. Полученный идентификатор необходимо указывать в запросах на открытие Payment Widget в значении параметрачтобы открывать платежную форму с набором свойств этого терминала., конфигурировании набора доступных платежных методов`terminal_id`
