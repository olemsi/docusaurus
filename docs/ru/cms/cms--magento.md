# Использование плагина cms_magento_plugin для CMS cms_magento

В данной инструкции описана работа с плагином cms_magento_plugin версии cms_magento_plugin_version. Этот плагин позволяет проводить оплаты и возвраты с использованием платежных решений Some Platform на сайтах, созданных на базе CMS cms_magento версии cms_magento_version или выше.

Скачать плагин cms_magento_plugin можно.

## Схема работы

Взаимодействие с платежными решениями Some Platform осуществляется через платежную страницу.

Запросы на открытие платежной страницы формируются автоматически (с помощью плагина) при выборе cms_paymentmethod в методах оплаты.

Если по какой-либо причине платеж не прошел, у пользователя есть возможность повторно ввести данные на платежной странице.

По результатам проведения платежей на сайт отправляются оповещения, которые также автоматически обрабатываются с помощью плагина. В соответствии с результатом платежа изменяется статус заказа:

- — платеж не прошел.**Pending Payment**
- — платеж прошел.**Processing**
- — заказ исполнен.**Complete**
- — полный возврат прошел.**Closed**
- — частичный возврат прошел.**Partial Refund**
