# Обзор

title — платежный метод, позволяющий пользователям осуществлять оплату с помощью банковских переводов. Оплата осуществляется через Payment Widget и Gate.

## Схема работы

При проведении платежа с использованием метода title задействуются веб-сервис мерчанта, один из интерфейсов и платежная платформа Some Platform, а также технические средства title, обеспечивающие взаимодействие с сервисами банков.
