# Использование

После запуска плагина он работает автономно.

Мы рекомендуем контролировать проведение заказов на вкладке, а также следить, чтобы номера заказов были уникальными в рамках сайта. При дублировании номера заказа платежная страница не открывается и заказ остается со статусом.**On hold**

Перед переводом плагина обратно в рабочий режим, убедитесь, что поляизаполнены значениями, полученными от службы технической поддержки Some Platform.**Project IDSecret key**

При возникновении вопросов по работе плагина обращайтесь в службу технической поддержки Some Platform по адресу support@example.com

::: danger
Если после запуска плагина вам потребуется переключить его в тестовый режим, обязательно имейте в виду описанную далее особенность. Если установлены флажкии, метод оплаты с помощью платежной страницы Some Platform отображается на сайте,**но все платежи и возвраты проводятся как тестовые!****Test mode**
:::
