## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей:

- Все платежи являются целочисленными. Округление выполняется до полной единицы в большую сторону.
- ru_qck_callbacks_fix.ditaЕсли пользователь выполнит перевод на сумму, отличную от суммы, указанной в запросе на оплату, проведение платежа будет выполняться с особенностями, которые описаны в разделе.
- В платежном методе LinkAja оплата может проводиться только через мобильное приложение. При попытке оплатить не с мобильных устройств пользователю будет предложен QR-код, отсканировав который пользователь попадет в мобильное приложение LinkAja.
- Оплата и выплата с использованием методов OVO Wallet и QRIS доступны только для мерчантов категории Financial Services (Forex, Binaries) по предварительному согласованию с курирующим менеджером Some Platform.
