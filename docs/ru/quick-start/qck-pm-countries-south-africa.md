## Особенности проведения платежей

При выплате в методе 1Voucher в оповещении дополнительно передается объект, содержащий параметры:с суммой на балансе ваучера и— с PIN-кодом ваучера. Эти данные пользователь должен использовать для получения выплаты в любом месте, где принимается 1Voucher — как онлайн, так и в супермаркетах.**provider_extra_fieldsamount_remnantpin**

## Выплата через банки

Выплата в методе Instant EFT осуществляется через банки, поддерживающие работу с этим методом. В запросе на выплату в параметревы должны передать код банка пользователя, который следует использовать для выполнения выплаты.
