В дополнение к обязательному полюв заголовке можно использовать любые другие поля из числа допустимых.`Host`[в HTTP версии 1.1](https://tools.ietf.org/html/rfc2616#page-31)

- Стартовая строка с указанием метода передачи запроса () и конечной точки, протокола и его версии (). Доступные конечные точки:`POSTHTTP/1.1`
- Заголовок с полем, содержащим доменное имя для запросов к платформе.`Host`
- Пустая строка, отделяющая служебную информацию от тела сообщения.
- Тело сообщения, содержащее JSON-строку в кодировке UTF-8 с набором данных и подписью к ним.
