## Подключение в Objective-C

Для подключения библиотек к проекту мобильного приложения необходимо:

1. `sdk_ios_frame_new`Перенести файлв папку проекта мобильного приложения.
2. Открыть цель (target) проекта.**GeneralEmbedded Binaries**Перейти в.**+**Щелкнуть.**Add Other**Щелкнуть.`sdk_ios_frame_new`Выбрать файл.**Add**Щелкнуть.**Build Settings**Выбрать раздел.**Always embed swift embedded librariesYes**Установить переключательв положение.Добавить библиотеку к проекту. В Xcode версии 12 для этого необходимо:
3. **NSCameraUsageDescriptionInfo.plist**`permission is needed in order to scan card`Добавить ключсо значениемв файл, чтобы предоставить пользователям возможность автоматического заполнения данных карты через ее сканирование.
4. **NSLocationWhenInUseUsageDescriptionInfo.plist**`fraud prevention`Если в мобильном приложении не используется запрос местоположения пользователя, добавить ключсо значениемв файл.Some Platform не запрашивает местоположение пользователя, если этого не делает мобильное приложение, но в соответствии с требованиями App Store значение ключа не должно быть пустым.Если мобильное приложение уже запрашивает информацию о местоположении пользователя, этот шаг можно пропустить.
5. **Privacy - Photo Library Usage DescriptionPrivacy - Photo Library Additions Usage DescriptionInfo.plist**Если мобильному приложению не предоставлено разрешение на сохранение данных на устройстве, необходимо добавить ключи ключс необходимыми значениями в файл. Указанные для данных ключей значения отображаются для пользователя при запросе разрешения на сохранение данных.
