# Разделы

В интерфейсе Dashboard находятся следующие разделы:

- **Приступить к работе**— с вашими последними действиями в Dashboard. Чтобы быстро переключаться между интересующими вас действиями, вы можете закрепить их в разделе, нажав на значок в форме сердца в правом верхнем углу действия.
- **Аналитика**— с предустановленным набором аналитических панелей для оперативного анализа проведения платежей и операций по вашим проектам (к которым есть доступ у используемой учетной записи).
- **Платежи**— с реестром всех проведенных и проводимых платежей, инструментами для настройки отображения данных в реестре и возможностями просматривать детальную информацию о платежах и выполнять одиночные возвраты.
- **Мануальные платежи**— с реестром всех проведенных и проводимых выплат, инструментами для настройки отображения данных в реестре и возможностью просматривать детальную информацию о выплатах, а также с инструментами для проведения одиночных и массовых выплат и массовых возвратов.
- **Ссылки на оплату**— с реестром всех проведенных и проводимых оплат по платежным ссылкам и инструментами для работы с платежными ссылками — их формирования, деактивации и автоматической отправки на адреса электронной почты пользователей.
- **Подписки**— с реестром зарегистрированных регулярных оплат (подписок), инструментами для настройки отображения данных в этом реестре и возможностями просматривать детальную информацию о регулярных оплатах, изменять их условия и отменять дальнейшие списания.
- **Справки**— с реестром справок, сформированных по определенным типам операций, и инструментами для экспорта сформированных справок.
- **Отчеты**— с инструментами для формирования отчетов и срезов на основе заданных условий и для просмотра и экспорта сформированных отчетов.
- **Финансы**— с информацией о текущих балансах мерчанта с разбиением по проектам.
- **Проекты**— с информацией об отдельных свойствах ваших проектов (к которым есть доступ у используемой учетной записи) и инструментами для настройки этих свойств.
- **Моя команда**— с реестром всех учетных записей ваших сотрудников и с инструментами для управления этими учетными записями и просмотра истории действий пользователей.
