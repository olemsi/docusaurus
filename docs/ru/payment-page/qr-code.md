# Оплата по QR-коду

## Общая информация

ru_pp_purchase.ditaВы можете запросить у Some Platform QR-код, чтобы пользователь по этому QR-коду мог перейти на страницу оплаты и оплатить. После того как пользователь откроет страницу оплаты, оплата будет проходить как описано в разделе.

## Шаги оплаты по QR-коду

Чтобы выполнить оплату с использованием QR-кода:

1. `domen_PP/payment/generate-qr`*pp_host*Здесь вместо <> надо указать доменное имя Payment Widget, которое следует узнавать у своего курирующего менеджера или в службе поддержки.Создайте и отправьте по URL-адресузапрос на оплату с использованием QR-кода.
2. Примите от платежной платформы ответ (response) в формате HTTP-сообщения с данными для отображения QR-кода пользователю.
3. Примите от платежной платформы оповещение (callback) с результатом проведения оплаты.

## Ответ

ru_pp_interaction_response_format.ditaПосле отправки запроса вы получите от платежной платформы ответ в формате HTTP-сообщения с данными для отображения QR-кода пользователю. Подробнее об ответах см..

`img_src`В ответе в параметресодержатся данные в формате base64, которые надо преобразовать в изображение с QR-кодом. Разместите это изображение на странице своего сайта или мобильного приложения, чтобы пользователь мог его просканировать и выполнить оплату.
