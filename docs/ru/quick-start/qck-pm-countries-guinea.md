## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей:

- Для проведения платежей доступны сервисы операторов мобильной связи Orange и MTN.
- Валютой платежа может быть только.references/ru/currencies/GNF.dita

## Оплата через сервисы мобильных операторов

Для отображения страницы оплаты с предварительно выбранным оператором мобильной связи или страны пользователя помимо параметратакже передайте в запросе объектс параметрами:

- — указатель предварительного выбора оператора мобильной связи, значение —;`1`
- — код страны пользователя в формате ISO 3166-1 alpha-2, значение для Гвинеи —;`GN`
- — идентификатор оператора мобильной связи, возможные значения:— для оператора Orange,— для MTN.`ORANGEMTN`

## Выплата через сервисы мобильных операторов

Выплата с использованием метода Mobile Money проводится через сервисы операторов мобильной связи, поддерживающие такие возможности. Чтобы провести выплату пользователю, передайте в запросе идентификатор оператора мобильной связи пользователя в параметре.

Ниже представлен список операторов мобильной связи и их идентификаторы для проведения выплаты:

- Orange —`ORANGE`
- MTN —`MTN`
