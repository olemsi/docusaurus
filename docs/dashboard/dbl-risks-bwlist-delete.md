# Deleting criteria

**Risks**You can delete criteria included in the whitelists and blacklists in thesection. Keep in mind that they should be deleted one by one.

1. **Black/White listsRisks**To go to the Criteria subsection, click thebutton in thesection.
2. Find the record you need using filters if necessary.
3. To delete the record, click thebutton in the corresponding line.
4. Make sure that the record has been deleted from the list.
