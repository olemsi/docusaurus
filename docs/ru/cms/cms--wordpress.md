# Использование плагина cms_wordpress_plugin для CMS cms_wordpress

В данной инструкции описана работа с плагином cms_wordpress_plugin версии cms_wordpress_plugin_version. Этот плагин позволяет проводить оплаты и возвраты с использованием платежных решений Some Platform на сайтах, созданных на базе CMS cms_wordpress версии cms_wordpress_version или выше с использованием плагина cms_wordpress_plugin_wc версии cms_wordpress_plugin_wc_version или выше.

Скачать плагин cms_wordpress_plugin можно.

## Схема работы

Проведение платежей осуществляется через платежную страницу Some Platform, а проведение возвратов — через консоль.при выборе`cms_wordpress_api_refund`

Запросы на открытие платежной страницы формируются автоматически (с помощью плагина) при выборе cms_paymentmethod в методах оплаты.

Если по какой-либо причине платеж не прошел, у пользователя есть возможность повторно ввести данные на платежной странице.

По результатам проведения платежей и возвратов на сайт отправляются оповещения, которые также автоматически обрабатываются с помощью плагина. В соответствии с результатом операции изменяется статус заказа:

- — платеж не прошел.**On hold**
- без изменения суммы заказа— платеж прошел.**Processing**
- с изменением суммы заказа — частичный возврат прошел.**Processing**
- — полный возврат прошел.**Refunded**
- — частичный возврат прошел.**Partially refunded**
