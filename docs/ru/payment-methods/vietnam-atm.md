# Обзор

title cards NAPAS — платежный метод, который позволяет осуществлять платежи с помощью карт, брендированных местным поставщиком платежных услуг NAPAS, который подключен к местным банкам. Обычные внутренние карты во Вьетнаме ограничиваются функциями выплаты заработной платы и снятия наличных, но с возможностью проведения внутренних покупок через платежную сеть NAPAS. Оплата осуществляется через Payment Widget и Gate.
