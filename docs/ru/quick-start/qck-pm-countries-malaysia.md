## Особенности проведения платежей

При проведении платежей с использованием платежного методаQR-код для оплаты действителен 5 минут. Если пользователь не выполнит необходимые для оплаты действия в течение этого времени, платеж отклоняется.`Shopee`

Особенности проведения платежей с использованием платежного метода:`Touch&amp;Go`

- Выплата с использованием методадоступна только для мерчантов категории Financial Services (Forex, Binaries) по предварительному согласованию с курирующим менеджером Some Platform.`Touch&amp;Go`
- Если пользователь не выполнит необходимые для оплаты действия в течение 15 минут после инициирования оплаты, платеж отклоняется.
- Если пользователь при оплате укажет неверную сумму, платеж останется в промежуточном статусе, пока конфликт не будет разрешен вручную техническими специалистами Some Platform.
- Если сумма в запросе на платеж совпадает с суммой одного из текущих платежей, то из-за особенностей работы провайдера платежная платформа может (но не обязательно делает это) увеличить сумму платежа на величину от 0,01 до 0,99 MYR. Это иногда требуется, чтобы измененная сумма была уникальной среди других находящихся в обработке платежей с той же суммой. Платформа избегает увеличения суммы платежа, если это возможно.
