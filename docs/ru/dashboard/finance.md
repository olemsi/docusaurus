# Контроль текущих и сводных балансов

## Общая информация

**Финансы**`FinanceMerchant Admin`При работе с Dashboard можно контролировать текущее состояние балансов и анализировать информацию по ним за выбранные периоды. Для этого в интерфейсе выделен раздел, доступ к которому регулируется отдельным правом и по умолчанию доступен учетным записям, отнесенным к ролями.

**Финансы**Далее в данном разделе представлена информация о балансах и разделе, описан порядок действий, необходимых для контроля балансов, и приведен список показателей, используемых при детализации информации о балансах за выбранный период.

::: tip
**Финансы**Информацию в разделеможно считать ориентировочной и использовать в ознакомительных целях, но ее не следует применять для итогового анализа и сверок. Это связано с тем, что при обработке информации о зачислениях и списаниях неизбежно возникают технические и организационные задержки, связанные с взаимодействием между провайдерами платежных решений, определением итоговых комиссий и прочими процедурами, а в Dashboard отображается вся имеющаяся на конкретный момент информация. И эта информация может отличаться от итоговой.Для получения итоговой информации за конкретные периоды следует использовать регулярные операционные отчеты от Some Platform, а для уточнения информации о текущем состоянии балансов можно обращаться к курирующему менеджеру.
:::

Информация о текущем состоянии балансов включает в себя суммарные показатели по каждой используемой валюте и перечень балансов. Для просмотра этой информации следует:

1. **ТекущийФинансы**Перейти в подразделраздела.
2. **INOUTIN/OUT**Перейти на нужную вкладку (,,) и убедиться, что в реестре отображается требуемая информация: в левой части — итоговые суммы по балансам в доступных валютах, а в правой — перечень балансов с информацией о них.
3. **Контракт №Контракт с**Для этого можно использовать выпадающие спискии, расположенные на панели управления. Они позволяют фильтровать данные по связи с контрактами и действуют по отношению к данным на всех вкладках.**Финансы**В случаях, когда один баланс относится к нескольким контрактам и часть этих контрактов соответствует условиям фильтрации, а другие нет, в разделеотображаются полные сведения по этим балансам.При необходимости, отфильтровать информацию.

## Анализ информации за период

Информация за период включает в себя итоговые суммы по каждой используемой валюте на начало и конец периода, а также информацию с разбивкой списаний и зачислений за период по разным сводным показателям: суммарно по балансам в одной валюте и раздельно по каждому балансу. Для просмотра этой информации следует:

1. **СводныйФинансы**Перейти в подразделраздела.
2. **Поиск**Для этого следует вызвать фильтр с помощью кнопки, задать необходимые условия и щелкнуть кнопку.**Финансы**В случаях, когда один баланс относится к нескольким контрактам и часть этих контрактов соответствует условиям фильтрации, а другие нет, в разделеотображаются полные сведения по этим балансам.Задать условия для отображения данных: период и при необходимости другие атрибуты.
3. **INOUTIN/OUT**сводка с условиями выборки и сводные показатели для суммарных данных по разным балансам в одной валюте;перечень балансов со сводными показателями для данных по каждому балансу.Перейти на нужную вкладку (,,) и убедиться, что в реестре отображается информация, соответствующая заданным условиям:
4. **Показать итоговые суммы**Для этого следует использовать переключатель.При необходимости, выбрать режим отображения информации: с отображением сводки и перечня балансов (используется по умолчанию) либо только сводки.

Далее приведен список этих показателей с указанием типов балансов, для которых они могут использоваться, и краткими описаниями. С любыми вопросами, касающимися этих показателей, можно обращаться к курирующему менеджеру Some Platform.
