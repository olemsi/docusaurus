## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей. Уточняйте у курирующего менеджера Some Platform, какие из следующих особенностей актуальны для вас:

- Пользователь должен совершить перевод средств в течение одного часа с момента отображения на платежной форме реквизитов для перевода, в противном случае платеж будет отклонен.
- Время проведения платежей может составлять до 24 часов из-за особенностей работы провайдера.
- ru_qck_callbacks_fix.ditaЕсли пользователь выполнит перевод на сумму, отличную от суммы, указанной в запросе на оплату, проведение платежа будет выполняться с особенностями, которые описаны в разделе.
