# Тестирование

## Общая информация

Для метода title доступно тестирование оплат через Payment Widget и Gate, а также возвратов и выплат через Gate.

Тестирование может выполняться в рамках тестового проекта, и для подключения и отключения этой функциональности необходимо обращаться к специалистам технической поддержки Some Platform
                                support@example.com

1. Отправить в платежную платформу корректный тестовый запрос на открытие Payment Widget.
2. `force_pm`Если в запросе не был указан метод— выбрать метод title на странице эмулятора.

Для проведения тестового возврата через Gate необходимо отправить в платежную платформу корректный тестовый запрос на возврат средств, полученных по ранее проведенной тестовой оплате, и принять оповещение с информацией о результате.
