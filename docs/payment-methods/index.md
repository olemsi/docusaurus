# Payment Methods

This section covers the payment methods supported by the Platform payment platform. Each method article includes an overview, supported currencies and countries, integration guides for Host-to-host API and Payment Widget, and Dashboard analysis instructions.

## Available methods

- [**Open Banking**](./openbanking) — a group of internet banking methods based on open banking protocols, allowing processing payments in euros and other European currencies through various banks across Europe.
  - Available in: [Austria](./austria), [Belgium](./belgium), [Estonia](./estonia), [Finland](./finland), [France](./france), [Germany](./germany), [Greece](./greece), [Hungary](./hungary), [Latvia](./latvia), [Lithuania](./lithuania), [Netherlands](./netherlands), [Norway](./norway), [Poland](./poland), [Portugal](./portugal), [Romania](./romania), [Spain](./spain), [Sweden](./sweden), [UK](./uk)
