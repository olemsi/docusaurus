# Страницы с результатами проведения платежа

## Общая информация

Страница отображает результат проведения и информацию о деталях платежа.

На странице макета можно настраивать оформление элементов интерфейса (все элементы являются дополнительными).

## Элементы интерфейса

- Номер платежа.
- Описание платежа.
- Дата платежа.
- Данные о банковской карте.
- Надпись со статусом, суммой и валютой платежа.
- Изображение статуса платежа.
- Кнопка возврата в проект.
