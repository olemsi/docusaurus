# Поддерживаемые типы платежных карт

Payment Widget поддерживает проведение платежей и предварительный выбор типа платежного метода по следующим типам платежных карт:

и American ExpressВ соответствии с рекомендациями платежных систем Visa, Mastercardот 2020 года в при проведении платежей с использованием Payment Widget поддерживается возможность проведения платежей по картам с истекшим сроком действия. При этом когда пользователь указывает в платежной форме данные карты с истекшим сроком действия, ему отображается соответствующее предупреждение, и платеж проводится в соответствии с решением эмитента. Для подключения данной возможности необходимо обращаться в службу технической поддержки Some Platform.
