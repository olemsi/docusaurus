# Анализ результатов платежей

При работе с платежными методами, которые предоставляет Some Platform, доступны различные способы анализа информации о платежах и операциях.

**Аналитика**Всю необходимую информацию можно получать и анализировать средствами Dashboard, в том числе с помощью аналитических панелей в разделе.

**Отчеты**Также Dashboard позволяет выгружать данные в формате CSV с помощью инструментов в разделе. При этом можно выполнять разовые и периодические выгрузки информации на локальный компьютер.

С любыми вопросами о возможностях анализа можно обращаться в службу технической поддержки Some Platform.
