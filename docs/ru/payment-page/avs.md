# Проверка Address Verification Service

## Общая информация

**Address Verification Service**(AVS) — это сервис, который позволяет вам проверить, действительно ли пользователь, совершающий платеж, является владельцем банковской карты. Проверка происходит путем сверки адреса, указанного пользователем при осуществлении платежа, с адресом владельца карты по данным банка-эмитента карты.

. В случае если полученные данные не проходят проверку, не переданы или не заполнены, вы получите соответствующие код и сообщение об отказе в проведении платежа. Результат проверки AVS передается в оповещении в параметре.

Существует две схемы работы с учетом AVS: передача полей AVS в каждом запросе или передача полей AVS в запросе на уточнение параметров, если в настройках проекта включен AVS для страны-эмитента карты.

::: tip
Требование по предоставлению полей AVS сохраняется даже если платеж осуществляется по токену или сохраненной карте.
:::

## Результаты проверки AVS

avs_resultВозможные коды параметраи соответствующие значения и описания приведены в таблицах ниже.

| Код | Значение | Описание |
| --- | --- | --- |
| Z, P, W | Частичное совпадение | Почтовый индекс пользователя совпадает, адрес — нет |
| A, B | Частичное совпадение | Адрес пользователя совпадает, почтовый индекс — нет |
| F, D, M, Y, X | Полное совпадение | Адрес и почтовый индекс пользователя совпадают |
| N | Полное несовпадение | Ни адрес, ни почтовый индекс не совпадают |
| U, S, C, G, I | Информация недоступна | Для текущего аккаунта информация об адресе недоступна, либо банк-эмитент не поддерживает AVS |
| R | Система недоступна | Система авторизации банка-эмитента на данный момент недоступна. Можно повторить попытку |

| Visa | F, X, D, M, Y, A, B, Z, P, W, N, G, U, S, C, I, R, U |
| --- | --- |
| MasterCard | X, Y, A, Z, W, N, U, S, R, U |
| American Express | Y, N, A, Z, U, S, R |
