# Анализ проведенных платежей

## Общая информация

**Аналитика**Dashboard предлагает ряд инструментов, с помощью которых вы сможете анализировать проведенные платежи, создавая диаграммы и таблицы на их основе. Все инструменты визуализации информации о платежах находятся в разделе.

**Аналитика**Для создания диаграмм и таблиц с данными по платежам в разделеиспользуются следующие подразделы:

Далее подробно рассказывается, как с помощью инструментов раздела Dashboard создавать диаграммы и таблицы.

- **Операции**— для создания диаграмм с данными по операциям;
- **Платежи**— для создания диаграмм с данными по платежам;
- **Операции в сводной таблице**— для создания сводных таблиц с данными по операциям;
- **In/out оборот**— для создания таблиц с данными по обороту входящих и исходящих сумм;
- **Топ 10**— для создания диаграмм с данными по десяти платежным методам/странам/банкам/причинам отказов;
- **Причины отказов**— для создания диаграмм с данными по всем возможным причинам отказов по операциям.

## Создание диаграмм и таблиц

**Аналитика**С помощью инструментов разделавы можете представлять в виде диаграмм и таблиц данные по проведенным платежам за определенный период.

Чтобы создать диаграмму или таблицу:

**Чарты**После определения параметров и выбора фильтров на панелиотобразится диаграмма или таблица, созданная согласно заданным критериям.

Вы можете открыть в полноэкранном режиме созданную диаграмму или таблицу, а также удалить, редактировать или скачать ее, щелкнув по кнопке со значкомв правом верхнем углу диаграммы или таблицы и выбрав соответствующий элемент списка.

1. **Аналитика**Перейдите в раздел.
2. **Создать диаграммуЧарты**Щелкните кнопкув правом верхнем углу или щелкните ссылку в центральной части панели.
3. В открывшемся окне выберите интересующий вас подраздел.
4. Задайте параметры создаваемой диаграммы или таблицы.
5. **Сохранить и продолжить**Выберите фильтры создаваемой диаграммы или таблицы, а затем щелкните кнопку.

::: tip
Если при создании диаграммы или таблицы не задать значения фильтрам, то по умолчанию диаграмма или таблица представит данные по всем проектам мерчанта, типам операций, платежным методам и валютам.
:::

## Создание диаграмм и таблиц по шаблонам

Вы также можете создавать диаграммы и таблицы по готовым шаблонам с уже заданными параметрами. Dashboard предлагает следующие шаблоны:

Чтобы создать диаграмму или таблицу по готовому шаблону, перейдите в соответствующий подраздел и щелкните кнопку в правом верхнем углу подраздела. При необходимости измените параметры шаблона, которые используются при создании диаграмма или таблица.

- **КонверсияОперации**— шаблон для создания диаграмм, представляющих соотношение успешных и неуспешных операций за прошедшие семь дней. Кнопка для создания диаграмм по этому шаблону доступна в подразделе;
- **ОборотIn/out оборот**— шаблон для создания таблиц с итоговой суммой по каналам валюты за прошедшие семь дней с учетом типа платежного метода и типа операций. Кнопка для создания таблицы по этому шаблону доступна в подразделе;
- **По эмитентамТоп 10**— шаблон для создания диаграммы с десятью банками-эмитентами, по которым прошло наибольшее количество операций за прошедшие семь дней. Кнопка для создания диаграммы по этому шаблону доступна в подразделе.
