# Страница ввода дополнительных данных

## Общая информация

При использовании более одного дополнительного параметра для сбора сведений о пользователе соответствующие параметрам поля отображаются на отдельной странице.

- Сообщение “Введите дополнительные данные”.

Месторасположение этих логотипов на странице можно менять.
