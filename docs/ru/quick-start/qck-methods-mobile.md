# Мобильная коммерция

«Мобильная коммерция» — это популярный платежный метод, с помощью которого пользователи могут оплачивать в вашей системе различные услуги, а также получать денежные средства.

Точную минимальную и максимальную сумму оплаты и выплаты уточняйте у курирующего менеджера Some Platform. Уточнить минимальную и максимальную сумму платежа, доступную в вашем проекте, вы также можете в Dashboard. Для этого в Dashboard перейдите в раздели выберите вкладку.**ПроектыПлатежные методы**

Логотип платежного метода доступен для загрузки.здесь

## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей:

- Доступность возврата необходимо уточнять у курирующего менеджера в Some Platform.
- При оплате пользователь получает от сервиса оператора связи сообщение с OTP-кодом для подтверждения оплаты.
