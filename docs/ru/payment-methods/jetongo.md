### Общая информация

Для оплаты через Gate с использованием метода title со стороны веб-сервиса необходимо:

Полная схема проведения оплаты представлена далее.

ru_gate_sts_protocol.ditaИнформация о формате запросов и параметрах инициализации оплат методом title через Gate, а также о формате оповещений о результатах оплат приведена далее, общая информация о работе с API см. в разделе.

Если у валюты нет дробных единиц (то есть количество разрядов дробных единиц равно нулю), то в этом параметре нужно указывать сумму в. Подробнее о разрядах дробных единиц у валют см.;currency— код валюты оплаты в формате ISO-4217 alpha-3.— объект, содержащий сведения об оплате:return_urlsuccess— URL для перенаправления пользователя при успешной оплате;decline— URL для перенаправления пользователя при отклонении оплаты.— объект, содержащий URL-адрес для перенаправления пользователя в веб-сервис:В запросе должны использоваться следующие объекты и параметры:
3. references/ru/currencies/EUR.ditareferences/ru/currencies/GBP.ditareferences/ru/currencies/INR.ditareferences/ru/currencies/JPY.ditareferences/ru/currencies/NOK.ditareferences/ru/currencies/PLN.ditareferences/ru/currencies/RUB.ditareferences/ru/currencies/TRY.ditareferences/ru/currencies/USD.ditaВалютой платежа могут быть:,,,,,,,,.
