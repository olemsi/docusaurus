# Общая информация

При работе с платежной платформой Some Platform для получения актуальной информации о выполнении запросов на программном уровне служат следующие сущности:

Также получать актуальную информацию о выполнении запросов можно через интерфейс Dashboard.

- Ответы отправляются синхронно, то есть в течение одного HTTP-сеанса, и содержат информацию о приеме запроса в платформе или об ошибке, если запрос не был принят.**Ответ (response)**
- Оповещения отправляются асинхронно и содержат информацию о промежуточном и итоговом результате выполнения запроса.**Оповещение (callback)**
