# Коды ошибок Gate при обработке запросов

*код ошибки*Если запрос был получен, но его обработка прекратилась из-за ошибки, то ответ, в
                дополнение к коду ответа, содержит.

| Код ошибки | Описание |
| --- | --- |
| 1004 | Валюта, указанная в запросе, не доступна для проведения платежей                                 в указанном проекте |
| 1005 | Неверная подпись запроса |
| 1006 | Переданная сумма платежа меньше минимально допустимой в                                 результате конвертации валют |
| 1007 | Истек срок действия указанной банковской карты |
| 1009 | Банковская карта не прошла 3D-Secure проверку |
| 1016 | Внутренняя ошибка |
| 1017 | Выплата не может быть осуществлена на основании переданного                                 платежа |
| 1020 | customerПараметрне передан при попытке                                 сохранения карты |
| 10102 | Указанный платеж не найден |
| 10105 | payment_idПлатеж с указанным номером () уже                                 существует |
| 10201 | Указанный пользователь не найден |
| 10309 | Попытка произвести рекуррентный платеж в отсутствие                                 зарегистрированных рекуррентных платежей |
| 124001 | Указанная карта не найдена |
