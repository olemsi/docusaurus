## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей:

- Если пользователь при оплате с использованием метода Bank Transfers укажет сумму, отличающуюся от суммы, указанной в запросе, оплата будет переведена в промежуточный статус. Для перевода платежа в один из конечных статусов обратитесь к специалистам технической поддержки Some Platform.
- Если пользователь при оплате с использованием метода Bank Transfers не выполнит необходимые для оплаты действия в течении 30 минут после инициирования оплаты, платеж отклоняется.
- В платежном методе PagoEfectivo срок действия ваучера по умолчанию составляет 5 дней. Для изменения срока действия ваучера обратитесь к специалистам технической поддержки Some Platform.
