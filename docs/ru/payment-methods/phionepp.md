# Выплата через Gate

## Общая информация

Для выплаты через Gate с использованием метода title со стороны веб-сервиса необходимо отправить запрос, содержащий обязательные параметры и подпись, на рабочий URL Some Platform и принять оповещение о результате выплаты. Полная схема проведения выплаты представлена далее.

ru_gate_sts_protocol.ditaИнформация о формате запросов и параметрах инициирования выплаты методом title через Gate, а также о формате оповещений о результатах выплаты приведена далее. Общую информацию о работе с Gate API можно найти в разделе.

## Формат запросов

При работе с запросами на выплату с использованием метода title необходимо учитывать следующее:

Вот пример данных из запроса на выплату с использованием метода title:

1. `/v2/payment//payout`Следует использовать POST-запрос к конечной точке.
2. generalproject_id— идентификатор проекта, полученный от Some Platform при интеграции;payment_id— идентификатор платежа, уникальный в рамках проекта;signatureИспользование подписи к данным— подпись запроса, составленная после определения всех параметров запроса. (Подробнее см. раздел.)— объект, содержащий основные сведения о запросе:customerid— идентификатор пользователя, уникальный в рамках проекта;ip_address— IP-адрес устройства пользователя;— объект, содержащий сведения о пользователе:paymentamount*за исключениемосновных единицах валюты*ru_currency_codes.dita— сумма выплаты в дробных единицах валюты без десятичной точки и пробеловслучаев, когда у валюты нет дробной части. Если у валюты нет дробных единиц (то есть количество разрядов дробных единиц равно нулю), то в этом параметре нужно указывать сумму в. Подробнее о разрядах дробных единиц у валют см.;currency— код валюты выплаты в формате ISO-4217 alpha-3.— объект, содержащий сведения о выплате:В запросе должны присутствовать следующие объекты и параметры:
3. Валютой выплаты может быть только currency.
4. Дополнительно могут использоваться любые другие (необязательные) параметры, указанные в спецификации.

## Формат оповещений

ru_gate_callbacks.ditaДля оповещений о результатах выплат с использованием метода title используется типовой формат, описание которого представлено в разделе.

`1234customer_123example_amount_in_majors,00&#160;example_currency`В следующем примере в оповещении содержится информация о том, что в рамках проектадля пользователяпроведена выплата в размере.

В следующем примере содержится информация об отклонении выплаты.
