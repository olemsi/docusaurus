# Оплата по токену

Payment Widget позволяет вам проводить оплату по токенам банковских карт.

Чтобы произвести оплату по токену, вам необходимо в запросе на открытие платежной страницы передать токен банковской карты пользователя. В таком случае пользователю откроется платежная страница с уже выбранной банковской картой и заполненными данными этой карты, кроме CVV или CVC. Пользователю остается только ввести CVV или CVC и подтвердить сумму платежа.

Пользователь не сможет выбрать другую банковскую карту или другой платежный аккаунт.

По любый вопросам, касающимся настройки работы токенов, обращайтесь к специалистам технической поддержки по адресу support@example.com
