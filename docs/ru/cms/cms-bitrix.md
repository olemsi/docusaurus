# Использование плагина cms_1cbitrix_plugin для CMS cms_1cbitrix

В этом разделе описана работа с плагином cms_1cbitrix_plugin версии.0. Этот плагин позволяет использовать платежные решения Some Platform на сайтах, созданных на базе CMS cms_1cbitrix в редакциях «Малый бизнес», «Бизнес» и «Энтерпрайз».21

## Схема работы

Взаимодействие с платежной платформой Some Platform осуществляется через платежную страницу.

Запросы на открытие платежной страницы формируются автоматически (средствам плагина), когда пользователь выбирает для оплаты заказа метод cms_1cbitrix_api_wl.

Если по какой-либо причине платеж не прошел, у пользователя есть возможность повторно ввести данные на платежной странице.

По результатам проведения платежей на сайт мерчанта в cms_1cbitrix отправляются оповещения, которые автоматически обрабатываются плагином. В соответствии с результатом платежа плагин изменяет статус заказа в cms_1cbitrix. Соответствие результатов платежей и статусов заказов настраивается в параметрах CMS cms_1cbitrix.

Плагин также поддерживает обратную логику: при изменении статуса заказа в cms_1cbitrix выполняются соответствующие действия в платежной платформе. Аналогично, в процессе двухстадийной оплаты платежи в статусеавтоматически отклоняются, когда заказ в cms_1cbitrix переводится в статус.`awaiting captureПодтверждение оплатыawaiting captureОтмена оплаты`
