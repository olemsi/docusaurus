# Страница Address Verification System

## Общая информация

Для ряда стран требуется проверка адреса и почтового индекса пользователя, которая осуществляется подключением проверки Address Verification Service (AVS). При подключении этой опции в составе макета должна быть проработанная AVS-страница. Страница открывается после ввода данных банковской карты.

ru_pp_avs.ditaС дополнительными сведениями можно ознакомиться в. На странице макета можно настраивать оформление элементов интерфейса.

## Обязательные элементы

- Почтовый индекс пользователя.
- Адрес пользователя.
- Кнопка оплаты, инициирующая отправку данных в платежную систему.
- Приложении 1Сообщения об ошибках. Отдельно должен быть проработан вид страницы с сообщениями об ошибках. Типовые тексты ошибок представлены в.
