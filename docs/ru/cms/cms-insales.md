# Приложение cms_insales_plugin для cms_insales

В этой статье описан процесс установки и настройки приложения cms_insales_plugin для CMS-системы cms_insales. Это приложение позволяет проводить платежи с использованием платежных решений Some Platform на сайтах, созданных на базе cms_insales.

## Схема работы

Взаимодействие с платежными решениями Some Platform осуществляется через платежную страницу.

Запросы на открытие платежной формы формируются автоматически с помощью приложения cms_insales_plugin, если оно было ранее установлено на сайте магазина в cms_insales и выбрано в качестве способа оплаты.

## Установка приложения  cms_paymentmethod

Перед началом установки необходимо создать магазин в платформе cms_insales. Далее приводятся инструкции по установке приложения на уже существующий магазин.

**Чтобы установить приложение cms_insales_plugin:**

1. Откройте магазин приложений cms_insales по адресу.[https://www.insales.kz/collection/apps](https://www.insales.kz/collection/apps)
2. Найдите приложение cms_insales_plugin, воспользовавшись поиском по ключевой фразе.`cms_insales_search`
3. Наведите курсор на карточку приложения, при наведении она изменит свой вид:
4. Щелчкомоткройте страницу установки приложения:**Установить**
5. Щелкнитеи подтвердите установку в открывшемся модальном окне.**Установить**

## Настройка приложения cms_paymentmethod

Чтобы полноценно пользоваться установленным приложением, выполните начальную настройку в следующем порядке:

При возникновении вопросов, связанных с установкой и работой приложения cms_insales_plugin для cms_insales, обращайтесь к специалистам технической поддержки Some Platform.

1. На странице административного интерфейса своего магазина в cms_insales перейдите в раздел, найдите приложение cms_insales_plugin и щелкните его.
2. Щелкните, если вы клиент Some Platform, в противном случае щелкните, чтобы перейти на страницу с инструкцией по интеграции.**ДаНет**
3. Введите данные, полученные от Some Platform при интеграции, в поля открывшейся формы (описание полей приведено ниже). После заполнения полей щелкните:**Сохранить**
4. На странице административного интерфейса своего магазина в платформе cms_insales перейдите в раздел, найдите приложение cms_insales_plugin и щелкните его.
5. Отметьте способы доставки, при выборе которых пользователю будет доступно проведение оплаты через платежную платформу Some Platform, и щелкните.**Сохранить**
