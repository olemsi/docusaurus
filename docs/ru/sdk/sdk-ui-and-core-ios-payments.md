# Проведение платежа

`Sale`По умолчанию в sdk_ui_ios настроено проведение разовой одностадийной оплаты (с типом действия). Для оплаты такого типа можно использовать приведенные выше примеры и дополнительно ничего не настраивать.

sdk_ui_ios также позволяет проводить и двухстадийные оплаты с блокировкой средств через SDK и последующим списанием. Для этого следует:

1. `AuthpaymentOptions`SwiftpaymentOptions.action = .AuthObjective-C[paymentOptions setAction: ActionTypeAuth];Вызовите платежную форму, задав тип действияв объекте:
2. Когда потребуется, подтвердите списание средств через Dashboard или через Gate. Чтобы подтвердить списание в Gate, отправьте запрос в конечную точку.
