# Рассрочка платежа

## Общая информация

При проведении оплаты с помощью платежных карт, вВслучае если пользователь не может оплатить всю сумму сразу, он может выполнить платеж с рассрочкой. Для этого на странице оплаты Payment Widget пользователь выбирает количество месяцев, в течение которых он совершит оплату по частям. Итоговая сумма первой части платежа, которая фактически списывается с карты, автоматически рассчитывается и отображается на Payment Widget пользователю. После получения запроса на списание первой части платежа с рассрочкой банк-эмитент или другая финансовая организация, выпустившая карту, оплачивает полную сумму платежа в платежную систему. Оставшиеся части платежа списываются банком-эмитентом с карты в течение выбранного пользователем периода.

Информацию о возможности подключения данной функциональности уточняйте в вашего курирующего менеджера Some Platform.

## Проведение платежа с рассрочкой

Для оплаты с рассрочкой со стороны веб-сервиса необходимо отправить запрос на открытие Payment Widget, содержащий требуемые параметры и подпись, на рабочий URL Some Platform и принять оповещение о результате оплаты.

ru_pp_parameters.ditaДополнительно могут использоваться любые другие параметры из числа доступных для работы с Payment Widget. Детальная информация обо всех параметрах приведена в разделе.

**1369121**customer_installment_countПосле выбора метода оплаты на странице ввода данных Payment Widget пользователь выбирает количество месяцев, в течение которых он внесет всю сумму платежа. Допустимые значения предоставляются пользователю в выпадающем списке:,,,,. При этом количество месяцев рассрочки можно сделать предварительно выбранным для пользователя, для этого в запросе на открытие необходимо передать параметр. По умолчанию устанавливается значение.

После выбора периода рассрочки на Payment Widget происходит автоматический перерасчет суммы оплаты, и с карты списывается только сумма первой части платежа.

ru_pp_callbacks.ditaДалее запрос на оплату с указанием полной суммы платежа и количества месяцев рассрочки передается от платежной платформы Some Platform в платежную систему. Платежная система передает данные платежа в банк-эмитент, получает от банка оплату на всю сумму платежа и отправляет уведомление о получении оплаты в платежную платформу Some Platform. Платежная платформа отправляет в веб-сервис оповещение с результатом оплаты и данными по рассрочке платежа, подробнее в разделе.

::: tip
*pp_host*Здесь вместо <> надо указать доменное имя Payment Widget, которое следует узнавать у своего курирующего менеджера или в службе поддержки.
:::
