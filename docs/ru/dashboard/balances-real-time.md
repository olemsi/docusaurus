# Контроль текущего состояния

Информация о текущем состоянии балансов включает в себя суммарные показатели по каждой используемой валюте и перечень балансов. Для просмотра этой информации следует:

1. **Финансы**Перейти в раздел.
2. **INOUTIN/OUT****ФинансыIN**Если в разделеотсутствует вкладка, следует обратиться к курирующему менеджеру Some Platform.Развернуть нужную вкладку (,,) и убедиться, что отображается требуемая информация: сверху — итоговые суммы по балансам в доступных валютах, а под ними — перечень балансов с информацией о них.

Для этого можно использовать выпадающий список с наименованиями контрактов, заключенных с Some Platform**Финансы**Когда один баланс относится к нескольким контрактам и только часть этих контрактов соответствует условиям фильтрации, в разделеотобразятся полные сведения по этим балансам.При необходимости, отфильтровать информацию по связи с контрактами.
