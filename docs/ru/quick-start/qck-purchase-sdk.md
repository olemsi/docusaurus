# Интеграция с использованием SDK

## Обзор

Интеграция с платежной платформой может выполняться с использованием SDK (Software Development Kit) — наборов средств разработки, предоставляемых вам в Some Platform.

В состав SDK входят библиотеки для определенных языков программирования, которые устанавливаются в проект с исходным кодом вашей системы вручную или автоматически с помощью компилятора. SDK позволяет из набора параметров формировать запрос на открытие страницы оплаты и получать информацию из оповещений (callbacks).
