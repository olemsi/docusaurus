## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей:

- В некоторых случаях, из-за особенностей работы провайдера, сумма платежа, приведенная в платежной инструкции, может незначительно отличаться от суммы в запросе на проведение платежа. Пользователь должен произвести оплату согласно платежной инструкции, при этом мерчанту зачисляется полная сумма, указанная в платежной инструкции, например:
- Если пользователь при оплате укажет неверную сумму, платеж останется в промежуточном статусе, пока конфликт не будет разрешен вручную техническими специалистами Some Platform.
