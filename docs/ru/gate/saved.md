## Получение списка сохраненных платежных данных

Дополнительно по запросу вы можете получить список сохраненных карт и других платежных инструментов пользователя и их данные; при этом номер карты передается в маскированном виде, CVV не передается.

## Создание запроса на получение списка сохраненных платежных средств

В запросе укажите идентификатор пользователя, проект и платежный метод.

account_idcard_idПосле окончания обработки запроса вам придет ответ со списком сохраненных платежных инструментов пользователя в выбранной платежной системе или без него, если нет ни одного сохраненного в системе платежного инструмента. Для каждого сохраненного платежного средства в параметреилиуказывается его идентификатор в Gate.

## Удаление сохраненных платежных данных

Если пользователь удаляет сохраненный платежный инструмент, отправьте запрос на удаление. Gate убирает такое платежное средство из списка сохраненных, но не удаляет его данные.

::: tip

:::
