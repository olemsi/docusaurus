## Особенности проведения платежей

`Bank transfers`**0050**Сумма в запросе на проведение платежа должна быть только целочисленной (например, 200,VES). Если в запросе передать нецелочисленную сумму (например, 200,VES), платеж будет отклонен на стороне провайдера.Если пользователь не выполнит необходимые для оплаты действия в течение 5 минут после инициирования оплаты, то платеж отклоняется.Указанные в инструкции реквизиты могут использоваться для оплаты только один раз. Для повторной оплаты пользователь должен инициировать новый платеж.Если пользователь выполнит перевод на сумму, отличную от суммы, указанной в запросе на оплату, оплата будет отклонена на стороне провайдера.Особенности проведения платежей с использованием платежного метода:
