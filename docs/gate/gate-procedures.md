# Auxiliary procedures

Some payments require additional platform-side procedures beyond the primary payment request. Gate supports these procedures as part of the overall processing workflow.

## In this section

- [**3-D Secure overview**](./gate-payment-3ds-overview) — customer authentication during card payments.
- [**Additional payment info**](./gate-clarification) — submitting extra data requested during processing.
- [**Address Verification (AVS)**](./gate-avs) — address-based cardholder verification.
- [**Currency conversion**](./gate-conversion) — handling payments involving different currencies.

## Scope

Use this section when a payment flow includes authentication, clarification, address verification, or conversion steps in addition to the main request.
