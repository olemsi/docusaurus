# Обработка оповещений

## Обзор

При работе с sdk_android поддерживается использование оповещений, отправляемых платежной платформой Some Platform напрямую к веб-сервису мерчанта, и оповещений, передаваемых вначале от платежной формы к sdk_android, а затем — от sdk_android к мобильному приложению мерчанта. В первом случае набор параметров, передаваемых в оповещении, может отличаться в зависимости от установленных настроек (подробнее:). Во втором случае передается строго определенный набор параметров — данные о статусе, типе и идентификаторе платежа, дате проведения платежа,сумме и валюте платежа:используемом платежном методе,ru_pp_callbacks.dita#ru_pp_callbacks/section_zlt_rgh_4bc

Далее представлена информация об этих оповещениях.

```
{
    "payment": {
        "status": "success",
        "type": "purchase",
        "id": "12345",
        "date": "2020-09-11T14:49:18+0000",
        "method": "card",
        "sum": 1000,
        "currency": "USD"
         }
}
```

## Оповещения платежной платформы

При проведении платежа платежная платформа отправляет оповещения на указанный URL. Этот URL необходимо сообщить службе технической поддержки Some Platform. Подробная информация об оповещениях представлена в разделе.ru_pp_callbacks.dita

## Оповещения sdk_android

Мобильное приложение мерчанта может принимать от sdk_android оповещения, которые передаются к sdk_android от платежной платформы. Оповещение отправляется мерчанту до того, как пользователю в платежной форме отображается страница с информацией о результате проведения платежа.

Информация о платеже передается через широковещательные сообщения, которые поступают в приложение мерчанта от sdk_android. Для получения этой информации мерчанту необходимо выполнить следующие шаги:

Далее представлены примеры реализациидля получения информации о результате проведения платежа.

1. Создать приемник широковещательных сообщений — объект.
2. При поступлении вшироковещательных сообщениях от sdk_android, с помощью ключаполучать из объектаинформацию о платеже. Ключ представляет собой константу, предоставляемую.`sdk_android_callback_type.getValue()sdk_android_activity`
