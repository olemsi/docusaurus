# Monitoring balances

Dashboard provides finance-related views for current balances, statements, and exchange-rate information. These views are intended for monitoring and operational analysis of financial data available in the platform.

## In this section

- [**Financial statements**](./dbl-finance-overview.md) — available finance tabs and statement-related tools.
- [**Balances overview**](./dbl-balances-aspects.md) — balance types, availability, and usage specifics.
- [**Analysing statement data**](./dbl-finance-statement.md) — how statement data is structured and reviewed.
- [**Retrieving balance data in real time**](./dbl-balances-real-time.md) — up-to-date balance retrieval scenarios.
- [**Currency rates**](./dbl-balances-currency-rates.md) — exchange-rate information used in finance workflows.

## Scope

Use this section for balance monitoring and statement analysis. For programmatic access to the same domain, see [**Using Data API**](./dbl-api-protocol.md).
