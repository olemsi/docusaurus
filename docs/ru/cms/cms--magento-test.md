# Тестирование

По умолчанию при активации плагина используется тестовый режим. Этот режим позволяет ознакомиться с работой плагина на сайте, к которому он подключен, без проведения реальных операций. В тестовом режиме можно настраивать и проверять работу плагина, проводить тестовые заказы и просматривать информацию по ним.

## Проведение тестовых заказов

После настройки параметров работы плагина следует провести тестовые заказы на сайте и проверить информацию по этим заказам в cms_magento, на вкладке.

::: tip
В тестовом режиме для изменения статуса заказа снапосле совершения платежа необходимо щелкнуть кнопку. В противном случае заказ остается со статусом.**Pending PaymentProcessingВернуться на сайтPending Payment**
:::
