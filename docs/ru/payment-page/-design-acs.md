# Страница Access Control System

## Общая информация

ACS-страница банка-эмитента может открываться двумя способами: в текущей или в новой вкладке. В текущей вкладке страница открывается перенаправлением, в новой вкладке — с помощью кнопки, инициирующей перенаправление на ACS-страницу банка-эмитента, при этом данная кнопка обязательно должна присутствовать на платежной форме, согласно политике безопасности браузеров.

Прорабатывать макет ACS-страницы не требуется, так как эта страница относится к банку-эмитенту.

## Элементы интерфейса

Кнопка для открытия ACS-страницы в новой вкладке. Можно менять расположение и оформление кнопки, которая обязательно должна быть проработана.
