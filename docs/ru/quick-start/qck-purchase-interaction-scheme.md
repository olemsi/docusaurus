# Схема проведения

Для проведения оплаты с помощью Payment Widget:

Порядок проведения оплаты — без выполнения дополнительных процедур — представлен далее на схеме.

1. Сформируйте и отправьте в платежную платформу запрос на открытие страницы оплаты.
2. Примите от платежной платформы оповещение о результате выполнения запроса.
