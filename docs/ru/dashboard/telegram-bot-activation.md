# Активация и деактивация бота telegram_bot

Перед первым использованием Telegram-бот telegram_bot нужно активировать в Dashboard, то есть предоставить учетной записи в Telegram доступ к Telegram-боту. Для этого выполняется привязка учетной записи Telegram к учетной записи в Dashboard и включение доступа к боту.

**Чтобы деактивировать бот telegram_bot(закрыть доступ к боту):**

Отключение учетной записи Dashboard влечет за собой автоматическую деактивацию бота для связанной учетной записи Telegram.

- **Мой профильTelegram ботАктивировать бота**Откройте разделв Dashboard и в разделеснимите флажок.

1. **Мой профильTelegram бот**Откройте разделв Dashboard и в разделещелкните ссылку, чтобы открыть диалог с Telegram-ботом telegram_bot.Если эта панель отсутствует в профиле учетной записи, обратитесь к сотрудниками технической поддержки Some Platform.
2. **НастройкиУзнать мой Telegram ID**Этот идентификатор также можно запросить через меню бота, выбрав.В открывшемся диалоге с Telegram-ботом скопируйте идентификатор учетной записи пользователя из приветственного сообщения бота.
3. **Telegram ботМой профильTelegram ID**Вернитесь в интерфейс Dashboard и на панелиразделавведите скопированный идентификатор в поле
4. **Активировать бота**Откроется окно с предложением ввести проверочный код.Включите флажок.
5. Вернитесь в диалог с Telegram-ботом и скопируйте поступивший в бот проверочный код.
6. Вернитесь в интерфейс Dashboard и введите скопированный код в окне для ввода проверочного кода.
7. Telegram-бот telegram_bot активирован и готов к работе.Убедитесь, что в диалоге с Telegram-ботом появилось сообщение об успешной активации бота.

::: warning
Если сотрудники мерчанта используют личные учетные записи Telegram для работы с ботом, то при увольнении таких сотрудников нужно обязательно отключать или удалять их учетные записи в Dashboard, чтобы у уволенных сотрудников не сохранялся доступ к Telegram-боту.
:::
