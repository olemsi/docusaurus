# Использование sts_api
        Gate

sts_api представляет собой единый интерфейс для работы с системой Some Platform. Этот интерфейс используется для проведения всех видов платежей, а также получения дополнительной информации. В этом разделе приведены общие сведения, необходимые для работы с sts_api.

sts_api основано на архитектуре REST API. Для передачи данных используется HTTP, при этом все данные передаются в формате JSON, а их проверка выполняется в соответствии с domen_contract. URL и параметры запросов приведены в domen_contract.
