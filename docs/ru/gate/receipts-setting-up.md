# Подключение

Подключение и настройка возможности отправлять уведомления пользователям выполняются специалистами технической поддержки Some Platform. При этом по согласованию с мерчантом настраиваются следующие характеристики режима отправки:

- перечень операций и других событий, для которых настраивается отправка;
- `successdecline`статусы операций, при которых выполняется отправка уведомлений (возможно как для итогового статуса, так и для итогового статуса) ;
- `customer.language`язык уведомления по умолчанию (русский или английский), который применяется в случае, если в запросе на проведение платежа не передан код языка пользователя в параметре;
- тема письма, в котором приходит уведомление (например, Receipt или «Уведомление о проведенной оплате»);
- адрес электронной почты отправителя (домен Some Platform или домен, принадлежащий веб-сервису мерчанта);
- возможность отправлять скрытые копии уведомлений на адрес электронной почты, предоставленный мерчантом.
