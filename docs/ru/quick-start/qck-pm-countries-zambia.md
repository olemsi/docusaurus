## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей с использованием платежного метода Mobile Money:

- Для проведения платежей доступны сервисы операторов мобильной связи MTN и ZAMTEL.
- Валютой платежа может быть только.references/ru/currencies/ZMW.dita
- Для валютысуммы в запросах указываются с дробной частью.references/ru/currencies/ZMW.dita

## Оплата через сервисы мобильных операторов

Для отображения страницы оплаты с предварительно выбранным оператором мобильной связи или страны пользователя помимо параметратакже передайте в запросе объектс параметрами:

- — указатель предварительного выбора оператора мобильной связи, следует передаватьпри выборе оператора связи или страны на стороне вашей системы;`1`
- — код страны пользователя в формате ISO 3166-1 alpha-2, значение для Замбии —;`ZM`
- — идентификатор оператора мобильной связи, возможные значения:— для MTN,— для ZAMTEL.`MTNZAMTEL`

Ниже представлен список операторов мобильной связи и их идентификаторы для проведения выплаты:
