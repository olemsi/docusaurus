# Общая информация

— это тип платежа, в рамках которого на основании исходного запроса осуществляется перевод денежных средств от пользователя в вашу систему.*Оплата*

Оплатав платежной платформе проводится с использованием Payment Widget. При проведении оплаты пользователь совершает базовые действия — выбирает платежный инструмент, указывает реквизиты, ожидает уведомление о результате платежа. В дополнение к базовым действиям может понадобиться выполнить обязательную процедуру дополнения информации о платеже.через страницу оплаты
