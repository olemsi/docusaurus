## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей:

- references/ru/currencies/KHR.dita21&#160;000,00&#160;— сумма, которая была указана в запросе на проведение платежа;references/ru/currencies/KHR.ditareferences/ru/currencies/KHR.dita21&#160;000,01&#160;— сумма, которая отображается в платежной инструкции. Пользователь должен совершить банковский перевод с указанием этой суммы, при этом мерчанту также будет зачислена эта сумма (21&#160;000,01&#160;).**00**references/ru/currencies/KHR.ditaВ случае, если пользователь совершит перевод с указанием иной суммы, например укажет первоначальную сумму платежа (21&#160;000,&#160;), проведение платежа приостанавливается, дальнейшие действия следует уточнять у специалистов технической поддержки Some Platform.В некоторых случаях, из-за особенностей работы провайдера, сумма платежа, приведенная в платежной инструкции, может незначительно отличаться от суммы в запросе на проведение платежа. Пользователь должен произвести оплату согласно платежной инструкции, при этом мерчанту зачисляется полная сумма, указанная в платежной инструкции, например:
- Если пользователь при оплате укажет неверную сумму, платеж отклоняется или остается в промежуточном статусе, пока конфликт не будет разрешен вручную техническими специалистами Some Platform. Актуальное для вас поведение уточняйте у курирующего менеджера Some Platform.
