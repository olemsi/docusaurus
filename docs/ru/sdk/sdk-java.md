## Установка и подключение библиотек

Устанавливать библиотеки, входящие в состав sdk_java, в проект с исходным кодом веб-сервиса можно вручную или автоматически. Способы установки и подключения библиотек могут отличаться в зависимости от среды разработки.

Чтобы установить библиотеки вручную и подключить их в исходном коде веб-сервиса, необходимо:

3. Подключить файл в проекте с исходным кодом веб-сервиса.
