# Dashboard

Dashboard is a web interface for managing payment solutions offered by the Platform payment platform. This interface allows its users to analyze detailed and consolidated payment processing data, perform payouts, issue refunds, and prepare reports.

Dashboard supports two-factor authentication for access according to the requirements mandated by PSD2, and access permissions are granted according to the role-based access model.

## Capabilities

- **Monitoring and analysing payment processing** — view, filter, search, and visualize data on both completed payments and payments in-progress.
- **Executing financial operations** — perform pay-ins, issue payouts and refunds (both full and partial).
- **Financial accounting and reporting** — prepare and view various reports with access to balance sheets.
- **Interface access administration** — manage user accounts and distribution of access roles, monitor user actions, and configure project settings.
