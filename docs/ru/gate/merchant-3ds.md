# Аутентификация 3ds на стороне мерчанта

## Общие сведения

Прохождение аутентификации 3ds пользователем может осуществляться как на стороне платежной платформы Some Platform, так и на стороне мерчанта. Если аутентификация проходит на стороне мерчанта, для совершения оплаты в платежную платформу необходимо передать результат прохождения проверки и повторная проверка на стороне платежной платформы происходить не будет.

Чтобы подключить эту возможность, необходимо обратитесь к специалистам технической поддержки (support@example.com
