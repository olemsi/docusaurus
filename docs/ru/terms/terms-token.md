# Токен

В платежной платформе Some Platform токены создаются для данных платежных карт и используются для безопасного обмена этими данными между Some Platform и мерчантами при проведении. Также для работы с statistics_api в платформе используются специализированные токены учетных записей.платежей

Также при работе с платформой могут использоваться и другие токены, формируемые сервисами платежных систем и провайдеров, например токены Apple Pay и Google Pay.
