## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей:

- Платеж отклоняется, если пользователь не совершит оплату в течение 8 минут с момента его перенаправления в сервис провайдера.
- Реквизиты для оплаты должны использоваться только один раз. Для повторной оплаты пользователь должен инициировать новый платеж.
