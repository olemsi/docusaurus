# Подключение

Для подключения плагина необходимо:

1. Открыть cms_opencart23.
2. Перейти на вкладку.
3. Щелкнуть кнопкуи выбрать файл с плагином (в форматеили).`ocmod.zipocmod.xml`**Загрузить**
4. Щелкнуть кнопку.**Продолжить**
5. Убедиться, что отобразилось сообщение:**Модуль успешно установлен!**<!-- image -->
6. Перейти на вкладкуи убедиться, что в списке модификаторов появился cms_opencart23_plugin.**Модификаторы**
7. Щелкнуть кнопку, чтобы обновить кеш модификатора.<!-- image -->
