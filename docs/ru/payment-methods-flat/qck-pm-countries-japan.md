## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей:

- references/ru/currencies/JPY.ditaДля валютысуммы указываются без дробной части.
- references/ru/currencies/JPY.dita21&#160;000&#160;— сумма, которая была указана в запросе на проведение платежа;references/ru/currencies/JPY.ditareferences/ru/currencies/JPY.dita21&#160;001&#160;— сумма, которая отображается в платежной инструкции. Пользователь должен совершить банковский перевод с указанием этой суммы, при этом мерчанту также будет зачислена эта сумма (21&#160;001&#160;).references/ru/currencies/JPY.ditaВ случае, если пользователь совершит перевод с указанием иной суммы, например укажет первоначальную сумму платежа (21&#160;000&#160;), проведение платежа приостанавливается, дальнейшие действия следует уточнять у специалистов технической поддержки Some Platform.В некоторых случаях, из-за особенностей работы провайдера, сумма платежа, приведенная в платежной инструкции, может незначительно отличаться от суммы в запросе на проведение платежа. Пользователь должен произвести оплату согласно платежной инструкции, при этом мерчанту зачисляется полная сумма, указанная в платежной инструкции, например:
- Если пользователь при оплате укажет неверную сумму, платеж останется в промежуточном статусе, пока конфликт не будет разрешен вручную техническими специалистами Some Platform.

## Выплата через банки

[https://bkichiran.hikak.com/](https://bkichiran.hikak.com/)[https://zengin.ajtw.net/](https://zengin.ajtw.net/)Проведение выплат в платежном методе Online banking осуществляется через банки, поддерживающие работу с этим методом. Для проведения выплаты, помимо других параметров, в запросе необходимо передавать название и код банка получателя, а также название и код филиала банка получателя. Для поиска и сверки информации о банках можно использовать специализированные реестры:с названиями и кодами банков получателей ис названиями и кодами филиалов этих банков.
