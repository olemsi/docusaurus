# Общая информация

— это тип платежа, в рамках которого на основании исходного запроса осуществляется перевод денежных средств от вашей системы к пользователю.*Выплата*

В платежной платформе поддерживается один вариант для работы с выплатой через Gate — разовая единичная выплата, но дополнительно обеспечивается возможность проведения массовой выплаты через Dashboard (с автоматическим формированием требуемого количества платежей).
