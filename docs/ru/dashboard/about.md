# Dashboard

Some PlatformDashboard — это интерфейс для управления платежными решениями, реализуемыми на базе платформы. С помощью этого интерфейса можно анализировать детальную и сводную информацию о проведении платежей, проводить оплаты, возвраты и выплаты, контролировать и предотвращать попытки мошенничества и выполнять иные действия. Dashboard предоставляет возможности для полноценной работы и вместе с тем активно развивается.
