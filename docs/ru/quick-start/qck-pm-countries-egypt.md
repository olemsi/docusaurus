## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей с использованием платежного метода Mobile Money:

## Оплата через сервисы мобильных операторов

Для отображения страницы оплаты с предварительно выбранным оператором мобильной связи или страны пользователя помимо параметратакже передайте в запросе объектс параметрами:

- — указатель предварительного выбора оператора мобильной связи, значение —;`1`
- — код страны пользователя в формате ISO 3166-1 alpha-2, значение для Египта —;`EG`
- — идентификатор оператора мобильной связи, возможные значения:— для оператора Etisalat,— для Orange,— для Vodafone,— для WePay.`ETISALATORANGEVODAFONEWEPAY`

## Выплата через сервисы мобильных операторов

Выплата с использованием метода Mobile Money проводится через сервисы операторов мобильной связи, поддерживающие такие возможности. Ниже представлен список операторов мобильной связи и их идентификаторы для проведения выплаты:

- Etisalat —`ETISALAT`
- Orange —`ORANGE`
- Vodafone —`VODAFONE`
- WePay —`WEPAY`
