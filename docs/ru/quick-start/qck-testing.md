# Тестовая среда

## Общая информация

Перед тем, как приступить к интеграции с платежной платформой Some Platform, вы можете познакомиться с процессом интеграции на примере тестовой среды. Для этого в платежной платформе настраивается специальная тестовая среда, а вы получаете идентификатор тестового проекта и секретный ключ от него.

Для проведения тестовых платежей в Some Platform доступны платежные методы, указанные в таблице ниже. Совершая тестовые платежи с использованием этих платежных методов, вы можете ознакомиться со всеми возможными сценариями платежа, встречающимися в процессе проведения реальных платежей.

- создавать запросы на открытие страницы оплаты;
- освоить предварительный выбор платежного метода на странице оплаты (подробнее см.);ru_qck_purchase_additional.dita#ru_qck_purchase_additional/section_wvv_c3w_5qb
- проверить возможности Payment Widget;
- ознакомиться со сценариями использования страницы оплаты с точки зрения пользователя;
- получать оповещения (callbacks) об успешном проведении платежа;
- получать оповещения (callbacks) о неуспешном проведении платежа;
- протестировать выплату.
