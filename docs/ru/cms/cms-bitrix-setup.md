# Подключение

## Установка плагина

Для подключения плагина необходимо выполнить следующие действия:

1. Загрузите архив с плагином cms_1cbitrix_plugin.
2. Откройте CMS cms_1cbitrix.
3. Перейдите на вкладку.
4. Добавьте папкув папку.`cms_1cbitrix_api_folderbitrix/modules`
5. Перейдите на вкладку.
6. Щелкнитенапротив cms_1cbitrix_plugin.**Установить**

## Прием оповещений

Автоматическое изменение платежного статуса заказа обеспечивается с помощью приема оповещений от Some Platform. Для получения оповещений необходимо:

1. Добавить файл форматав раздел.`php`<!-- image -->
2. Сообщите службе технической поддержки Some Platform URL-адрес для приема оповещений.

## Установка плагина

1. Загрузите архив с плагином cms_1cbitrix_plugin.
2. Откройте CMS cms_1cbitrix.
3. Откройте папку с подключаемыми модулями cms_1cbitrix.
4. Добавьте папкув папку.`cms_1cbitrix_api_folderbitrix/modules`<!-- image -->
5. Извлеките содержимое архива с плагином в папкув следующей последовательности:`bitrix/modules/cms_1cbitrix_api_folder`<!-- image -->
6. Подключите установленный плагин.

## Настройка приема оповещений

Автоматическое изменение платежного статуса заказа обеспечивается с помощью приема оповещений от Some Platform.

Чтобы обеспечить прием оповещений:

1. Откройте вкладкуи перейдите в раздел.**Контент**
2. Создайте файл приемщика оповещений.
3. Сообщите службе технической поддержки Some Platform URL-адрес для приема оповещений.
