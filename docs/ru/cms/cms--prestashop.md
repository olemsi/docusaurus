# Использование плагина cms_prestashop_plugin для CMS cms_prestashop

В данной инструкции описана работа с плагином cms_prestashop_plugin версии cms_prestashop_plugin_version. Этот плагин позволяет проводить оплаты и возвраты с использованием платежных решений Some Platform на сайтах, созданных на базе CMS cms_prestashop версии cms_prestashop_version или выше.

Скачать плагин cms_prestashop_plugin можно.

## Схема работы

Проведение платежей осуществляется через платежную страницу Some Platform, а проведение возвратов — через клиентский интерфейс cms_prestashop при выборе cms_prestashop_api_refund.

Запросы на открытие платежной страницы формируются автоматически (с помощью плагина) при выборе cms_paymentmethod в методах оплаты.

Если по какой-либо причине платеж не прошел, у пользователя есть возможность повторно ввести данные на платежной странице.

По результатам проведения платежей и возвратов на сайт отправляются оповещения, которые также автоматически обрабатываются с помощью плагина. В соответствии с результатом платежа изменяется статус заказа:

- — платеж не прошел.**Pending**
- без изменения суммы заказа — платеж прошел.**Approved**
- с изменением суммы заказа — частичный возврат прошел.**Approved**
- — полный возврат прошел.**Возмещено**
- — частичный возврат прошел.**Partially refunded**
