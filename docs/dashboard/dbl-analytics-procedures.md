# Using data visualisation tools

## Creating charts and tables

The procedure of adding a new chart or a table to your customised data visualisation board is the same for all datasets available in the section and implies configuring criteria of visualisations that are added. The configuration tool shows a relevant example of a chart or a table for each dataset.

To add a new chart or a table to the board:

1. **AnalyticsAdd chart**Navigate to thesection and clickin the top right corner of the section.Open the configuration tool.
2. Select the dataset you need on the left of the window that opens.
3. The set of required parameters depends on the type of the dataset. As a rule, required parameters include the title of a chart or a table, a time period, data type, and the chart type.Specify the parameters.
4. **Save and continue**Save changes by clickingin the bottom right corner of the window.
5. There will be a pop-up window with a notification that a new chart or table was added. You can also find it on the board, point with a mouse over theicon in the top right corner of the tile and confirm that this is the correct data.Make sure that the created visualisation the data you need visualisation for.

## Interacting with charts and tables

When you work with the tiles on the board, you can use the following functions of the tile:

1. Moving the chart or table across the board.
2. Configuring the chart or table and expanding the context menu.
3. Editing or deleting the chart or table.
4. Viewing in full-screen mode.
5. Exporting visualised data in different formats.

## Changing the layout of the board

**Edit layout**Tiles with visualisations on the board are laid out in one or two columns (with the two-column layout being the default option). To change the layout, click thebutton in the top right corner of the section and select the option you need.

In addition, you can drag the tiles across the board by placing the cursor on the dots located in the top left corner of the tile.
