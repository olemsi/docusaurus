# Тестовые карты

Для проверки корректности технической интеграции и отладки различных пользовательских сценариев для платежей с прямым использованием платежных карт рекомендуется проводить соответствующие тестовые платежи — в рамках тестовых проектов и с использованием тестовых карт.

Тестовые карты следует использовать только для тех сценариев, для которых те или иные карты предназначены. Остальные параметры, такие как сумма платежа, имя держателя карты или проверочный код, могут иметь различные значения, но должны указываться в корректном формате (в том числе это относится и к сроку действия карты, который должен заканчиваться позднее даты платежа).

подробнееВместе с возможностями тестирования платежей с прямым использованием карт, в платформе поддерживаются также возможности тестирования альтернативных платежных методов ().С любыми вопросами о тестировании платежей обращайтесь к специалистам технической поддержки Some Platform.
