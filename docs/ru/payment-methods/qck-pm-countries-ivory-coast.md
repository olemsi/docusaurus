## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей с использованием платежного метода Mobile Money:

- Для проведения платежей доступен сервис оператора мобильной связи MTN.
- references/ru/currencies/XOF.ditaВалютой платежа может быть только.
- references/ru/currencies/XOF.ditaДля валютысуммы в запросах указываются без дробной части.

## Оплата через сервисы мобильных операторов

force_payment_methodpayment_methods_optionsДля отображения страницы оплаты с предварительно выбранным оператором мобильной связи или страны пользователя помимо параметратакже передайте в запросе объектс параметрами:

- enable_mobile_service`1`— указатель предварительного выбора оператора мобильной связи, значение —;
- selected_country`CI`— код страны пользователя в формате ISO 3166-1 alpha-2, значение для Кот-д'Ивуар —;
- selected_operator`MTN`— идентификатор оператора мобильной связи, значение для оператора MTN —.

## Выплата через сервисы мобильных операторов

`MTN`account.service_providerВыплата с использованием метода Mobile Money проводится через сервис оператора мобильной связи MTN. Чтобы провести выплату через сервис MTN, передайте в запросе идентификаторв параметре.
