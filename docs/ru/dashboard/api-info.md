# Схема работы

При использовании statistics_api взаимодействие между сервисом мерчанта и платежной платформой Some Platform строится на обмене сообщениями HTTP по принципу «запрос-ответ» — с запросами от сервиса и ответами от платформы. При этом применяется синхронная схема взаимодействия, в рамках которой в ходе одного сеанса HTTP обрабатывается один запрос и отправляется однократный ответ на него, с запрошенной информацией или с информацией об ошибке.

(например, о балансах по проектам мерчанта)Такая схема работы обусловлена тем, что все запросы, принимаемые через statistics_api, подразумевают выполнение на стороне платежной платформы, без задействования других сервисов и систем. Вместе с тем, с учетом сложности запросов и объема подготавливаемых данных, время выполнения одного запроса на стороне платформы может существенно варьироваться: от десятков миллисекунд до нескольких минут (как правило, не более пяти).
