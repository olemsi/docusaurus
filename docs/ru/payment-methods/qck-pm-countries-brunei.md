## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей:

- references/ru/currencies/BND.ditaДля методов Cash in Brunei и Online banking: В качестве валюты оплаты можно использовать только.
- Для метода Cash in Brunei: Если пользователь при оплате укажет неверную сумму, платеж отклоняется или остается в промежуточном статусе, пока конфликт не будет разрешен вручную техническими специалистами Some Platform. Актуальное для вас поведение уточняйте у курирующего менеджера Some Platform.
- Для метода Cash in Brunei: Платеж отклоняется, если пользователь не совершит оплату в течение 2 часов с момента его перенаправления в сервис провайдера.
- Для метода Cash in Brunei: Получение выплаты в этом методе возможно как на точке получения наличных, так и с помощью доставки. Выбор способа получения наличных возможен с помощью двух конечных точек.
