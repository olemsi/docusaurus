# Запрос доступных платежных методов

Разные платежные методы поддерживают разные валюты, а также у платежных методов могут отличаться лимиты на сумму платежа. Кроме того, лимиты и поддерживаемые валюты могут меняться без предупреждения. Поэтому очень важно точно знать, какие методы доступны в проекте. Gate API позволяет получить перечень доступных методов, а также выяснить доступность конкретного метода для определенной валюты и суммы платежа.

Здесь рассказывается, как отправить запрос на получение списка доступных платежных методов, а также о том, как извлечь из полученного ответа информацию о доступных методах.

## Сравнение запросов

## Запрос на получение списка доступных платежных методов

В этом разделе приведены основные характеристики и базовые параметры запроса доступности методов в проекте.

payment_method_listЕсли в исходном запросе массивпустой или отсутствует, то в ответе вернется информация о доступности для всех платежных методах проекта.

references/ru/currencies/USD.ditaВот пример запроса доступности метода Apple Pay при оплате 100:

```json
{
  "general":{
    "project_id": 29791,
    "signature": "afb7db..."
    },
  "payment_method_list": [
        {
            "payment_method":"apple_pay_core",
            "amount": 10000,
            "currency", "USD"
        }
        ]
}
```

## Ответ с информацией о доступных платежных методах

Получив запрос, платежная платформа Some Platform синхронно возвращает ответ с информацией о доступности указанных в запросе платежных методов. В следующей таблице приводятся параметры, которые могут присутствовать в ответе.

Далее приведен пример ответа на приведенный в предыдущем разделе запрос. Ответ подтверждает доступность метода Apple Pay:

payment_method_listЕсли в запросе указывались конкретные платежные методы, то ответ будет содержать информацию только об эти методах. Если в исходном запросе массивпустой или отсутствует, то в ответе вернется информация о доступности для всех платежных методах проекта.
