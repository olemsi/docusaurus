# Параметры открытия платежной страницы Payment Widget

В этом разделе собраны все параметры, которые могут использоваться при вызове Payment Widget. Для каждого параметра приведено описание, а также информация о его имени, типе и обязательности.

Все параметры делятся на три вида по их обязательности:

- type_strictly_required — параметр всегда обязателен
- type_optional — параметр необязателен
- type_conditional — обязательность параметра зависит от ситуации. Подробнее узнавайте в описании параметра.
