## Особенности проведения платежей

Если пользователь при оплате укажет неверную сумму, платеж будет отклонен.

Особенности проведения платежей с использованием платежных методови:`KBZPayWave Money`

- Если пользователь при оплате укажет сумму, отличную от суммы, указанной в запросе на оплату, то платеж останется в промежуточном статусе, пока конфликт не будет разрешен вручную техническими специалистами Some Platform.
- Реквизиты для оплаты должны использоваться только один раз. Для повторной оплаты пользователь должен инициировать новый платеж.
- Время проведения платежа ограничено. Если пользователь не выполнит необходимые для оплаты действия в течение 10 минут после перенаправления в сервис провайдера, то платеж отклоняется.
