# Получение истории платежей пользователя

## Получение истории платежей пользователя

Gate хранит полную историю платежей пользователей проекта. При необходимости, вы можете запросить и получить историю платежей любого пользователя.

## Создание запроса на получение истории платежей пользователя

В запросе укажите идентификатор проекта и идентификатор пользователя.

В оповещении вы получите список платежей, совершенным указанным пользователем в указанном проекте.
