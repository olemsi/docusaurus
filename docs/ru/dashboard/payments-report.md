# Создание отчетов

## Общая информация

В Dashboard вы можете создавать отчеты по платежам, проведенным с помощью платежной платформы Some Platform. При этом готовые отчеты вы можете скачивать на свое устройство в формате CSV.

**Отчёты**Все инструменты, необходимые для создания отчетов, находятся в разделе.

## Создание отчета

Чтобы создать отчет по проведенным платежам в Dashboard:

**Сохранить как шаблонМои шаблоны**Вы можете сохранять параметры и фильтры отчета в качестве шаблона, поставив флажок рядом с обозначениемв левом нижнем углу интерфейса. Сохраненный шаблон появится на панелипри создании нового отчета.

**в обработке**На подготовку сохраненного отчета потребуется некоторое время. Пока отчет подготавливается, он будет находиться в статусе.

**успешноСкачать отчетОтчёты**Когда отчет будет готов, его статус сменится на, и вам на почту придет электронное письмо с предложением скачать готовый отчет. Вы можете скачать отчет, щелкнув кнопкув электронном письме или кнопкув разделев Dashboard.

1. **Отчёты**Перейдите в раздел.
2. **Создать**Щелкните кнопкув левом верхнем углу интерфейса.
3. Задайте обязательные параметры и колонки создаваемого отчета.
4. **Сохранить**Выберите интересующие вас фильтры, если необходимо. Щелкните кнопку.

::: tip
Обратите внимание, что отчет будет доступен для скачивания только в течение четырех дней с момента его создания.
:::
