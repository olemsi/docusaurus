### Общая информация

Для оплаты через Gate с использованием метода title со стороны веб-сервиса необходимо:

Полная схема оплаты выглядит следующим образом.

ru_gate_sts_protocol.ditaИнформация о формате запросов и параметрах инициирования оплаты методом title через Gate, а также о формате оповещений о результатах оплаты приведена далее. Общую информацию о работе с Gate API можно найти в разделе.
