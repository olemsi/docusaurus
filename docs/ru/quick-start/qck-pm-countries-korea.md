## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей:

- Для валютысуммы в запросе указываются без дробной части.references/ru/currencies/KRW.dita
- В платежном методе Virtual Accounts при выполнении пользователем оплаты в первый раз ему будет предложено зарегистрироваться в сервисе провайдера. При последующих оплатах передаваемые в запросе данные пользователя должны совпадать с теми, которые пользователь указал при регистрации, в противном случае пользователь не сможет выполнить оплату. Для изменения данных пользователя обратитесь к специалистом технической поддержки Some Platform.
- В платежном методе Virtual Accounts, если пользователь не выполнит необходимые для оплаты действия в течение 20 минут после инициирования оплаты, то платеж будет отклонен по истечении предельного времени проведения платежа.

1. Зарегистрироваться на сайте мерчанта.
2. Зарегистрироваться на сайте провайдера.
3. Пройти процедуру KYC (Know Your Customer) на сайте провайдера.
4. Связать на сайте провайдера код мерчанта и адрес электронной почты пользователя.
