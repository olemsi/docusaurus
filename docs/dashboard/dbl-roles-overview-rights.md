# Access rights

For example, to ensure that your employees have access to editing whitelists, you need to request a corresponding permission.Access rights to work with different sections of Dashboard are distributed among the roles as shown in the following table. In addition to these rights, you can request a number of permissions with the extended scope from the Platform technical support—on condition that such permissions are used in the platform and are available for the use by the merchant.
