## Оплата через Gate

### Общая информация

Для оплаты через Gate с использованием метода title со стороны веб-сервиса необходимо:

ru_gate_sts_protocol.ditaИнформация о формате запросов и параметрах инициирования оплат через Gate при работе с методом title, а также о форматах данных для перенаправления пользователей и о формате оповещений о результатах оплат приведена далее. Общая информация о работе с API — в разделе.

1. Отправить запрос, содержащий требуемые параметры и подпись, на рабочий URL Some Platform.
2. Осуществить перенаправление пользователя к форме оплаты title.
3. Принять оповещение о результате оплаты.
