# Страница ожидания

## Общая информация

Во время передачи данных и ожидания подтверждения платежа пользователю отображается страница ожидания. На странице можно менять оформление элементов интерфейса (все являются обязательными).

## Элементы интерфейса

- Анимация или рисунок загрузчика.
- Надпись с текстом загрузчика.
