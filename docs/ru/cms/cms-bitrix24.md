# Приложениедля cms_bitrix24**cms_bitrix24_plugin**

В этой статье описан процесс установки и настройки приложениядля CMS-системы cms_bitrix24. Это приложение позволяет проводить платежи с использованием платежных решений Some Platform на сайтах, созданных на базе cms_bitrix24.**cms_bitrix24_plugin**

## Схема работы

Взаимодействие с платежными решениями Some Platform осуществляется через платежную страницу Payment Widget. Подробнее о Payment Widget см..ru_pp_about.dita

Запросы на открытие платежной формы формируются автоматически средствами приложения, если оно было ранее подключено на сайте магазина в cms_bitrix24.**cms_bitrix24_plugin**

## Подключение и настройка приложения**cms_bitrix24_plugin**

После завершения подготовки можно переходить к настройке приложенияна своем сайте.**cms_bitrix24_plugin**

**Чтобы подключить ПСдля оплаты на сайте:**cms_bitrix24_plugin****

При возникновении любых вопросов по установке и работе приложениядля cms_bitrix24 обращайтесь к специалистам технической поддержки Some Platform.**cms_bitrix24_plugin**

1. Войдите в личный кабинет на сайте cms_bitrix24 по адресу.[https://bitrix24.kz](https://bitrix24.kz)
2. В левой панели последовательно выберите.
3. На странице приложений выполните поиск по строкеи найдите приложение.**cms_bitrix24_plugin**`Some Platform`
4. Щелкните, чтобы открыть страницу информации о приложении.**Посмотреть**
5. Щелчкомоткройте диалоговое окно установки со списком прав, которые требуются приложению.**cms_bitrix24_plugin****Установить**

7. Дождитесь окончания установки приложения и открытия окна с предложением настроить приложение cms_bitrix24_plugin.
8. В окне с предложением настроить приложение cms_bitrix24_plugin щелкните.**Да**
