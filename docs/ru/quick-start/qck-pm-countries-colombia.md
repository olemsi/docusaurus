## Особенности проведения платежей

Нужно помнить про некоторые особенности проведения платежей:

- Если пользователь при оплате с использованием методов Bank Transfers или Bre-B укажет сумму, отличающуюся от суммы, указанной в запросе, оплата будет отклонена на стороне провайдера.
- В платежном методе Efecty срок действия ваучера ограничен. Уточняйте срок действия ваучера у курирующего менеджера в Some Platform.
- Если пользователь при оплате с использованием метода PSE укажет сумму, отличающуюся от суммы, указанной в запросе, платеж останется в промежуточном статусе, пока конфликт не будет разрешен вручную специалистами технической поддержки Some Platform.
- При оплате с использованием метода Bre-B реквизиты для оплаты можно использовать только один раз. Для повторной оплаты пользователь должен инициировать новый платеж.
- Для использования платежной системы Bre-B пользователю необходимо установить приложение Bre-B и открыть в нем счет.
- Платеж с использованием метода Bre-B будет отклонен, если пользователь не совершит оплату в течение 5 минут с момента инициирования платежа.
