## Подключение библиотек в Swift

Для подключения библиотек к проекту мобильного приложения необходимо:

1. `sdk_ios_frame_framework`Перенесите файлв папку проекта мобильного приложения.
2. Откройте цель (target) проекта.**GeneralEmbedded Binaries**Перейдите в.**+**Щелкните.**Add Other**Щелкните.`sdk_ios_frame_framework`Выберите файл.**Add**Щелкните.Добавьте библиотеку в проект. В Xcode версии 12 для этого необходимо:
3. **NSCameraUsageDescriptionInfo.plist**`permission is needed in order to scan card`Добавьте ключсо значениемв файл, чтобы предоставить пользователям возможность автоматического заполнения данных карты через ее сканирование.
4. **NSLocationWhenInUseUsageDescriptionInfo.plist**`fraud prevention`Если в мобильном приложении не используется запрос местоположения пользователя, добавьте ключсо значениемв файл.Some Platform не запрашивает местоположение пользователя, если этого не делает мобильное приложение, но в соответствии с требованиями App Store значение ключа не должно быть пустым.Если мобильное приложение уже запрашивает информацию о местоположении пользователя, этот шаг можно пропустить.
5. **Privacy - Photo Library Usage DescriptionPrivacy - Photo Library Additions Usage DescriptionInfo.plist**Если мобильному приложению не предоставлено разрешение на сохранение данных на устройстве, добавьте ключиис необходимыми значениями в файл. Значения этих ключей отображаются пользователю при запросе разрешения на сохранение данных.
